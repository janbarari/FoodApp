# FoodApp
 ![](foodapp.gif)
 
This project fetches information form an api and uses the following technologies:


  - Kotlin
  - Clean Architecture
  - Design Patterns (Singleton, Factory, Repository, Strategy, Observer)
  - UI Transition
  - MVVM
  - Room
  - Coroutines
  - Retrofit
  - Android Navigation Components
  - LiveData
  - ViewModel
  - Kodein (Dependency Injector)
  - Unit Test
  - UI Test (Espresso)
  - Firebase
